File Format Guide
Create a .txt file with this exact format, then upload it.

Basic Structure

QUIZ: Your Quiz Title Here Q: Question text goes here? A: First option B: Second option C: Third option D: Fourth option ANS: B Q: Another question? A: Option one B: Option two C: Option three D: Option four ANS: A
Rules:
• Start with QUIZ: followed by the quiz name
• Each question starts with Q:
• Options are A: B: C: D: (2–4 options supported)
• Correct answer is ANS: followed by the letter
• Blank lines between questions are optional
• You can have multiple quizzes in one file — just start a new QUIZ: block

Multi-Quiz Example

QUIZ: Math Basics Q: What is 2 + 2? A: 3 B: 4 C: 5 D: 6 ANS: B Q: What is 10 / 2? A: 3 B: 4 C: 5 D: 6 ANS: C QUIZ: Science Facts Q: What planet is closest to the Sun? A: Earth B: Venus C: Mercury D: Mars ANS: C
